扬帆起航

这几天突然开始想整一下个人博客，在github上发现其实几年前自己已经配置过一个Github Pages，于是重新将这个仓库拾起，进行了装修。

利用Github Pages可以搭建一个免费的个人博客，也省去了去aliyun上购买云主机，注册域名之类的操作，如果只是日常记录一些内容，完全够用了。

自己的仓库是几年前用hugo生成的，在CSDN上搜了一篇文章用hexo搭建，并配合fluid主题，于是操作了一番，便产生了这个网页

今天学了一些东西

  1. 利用hexo搭建个人博客网站
    在Github上创建对应的仓库xxx.github.io,设置pages，可以指定不同的分支，原来使用的是master，现在也切换到另外一个分支了

  2. 利用Hexo初始化了一个项目，结合Fluid主题,个人感觉比之前的hugo风格好看点

  3. 开始利用Markdown写作，开始尝试一些标签，作为一个技术人，还是需要掌握这些东西的

在Visual Code中安装插件 Markdown all in one Markdown Preview Github Styling 可以直接实时预览文字效果

  1. 图片需要尝试使用下

忘了从哪里看到，开始写博客，不知道写什么内容，就写一下TIL，也就是Today I Learn，今天我学了什么，那么就用这些文字记录下吧。

希望能逐步提高写作的水平。
